# Contributing to Endeavor

Thank you for your interest in contributing to **Endeavor**.

## How to Contribute

- Fork the repository
- Create a focused branch for your change
- Keep pull requests small and easy to review
- Use clear commit messages and descriptions
- Update documentation when your change affects project usage or structure

## Good First Contributions

New contributors can help by:

- improving README clarity,
- refining project structure,
- adding setup instructions when source files are available,
- cleaning up naming or documentation,
- proposing beginner-friendly issues for future contributors.

## Pull Request Tips

- Explain **what** changed and **why**
- Avoid unrelated edits in the same pull request
- Reference issues when applicable
- Be respectful and collaborative during review

## Questions

If you are unsure where to start, open an issue describing the area you would like to improve.
